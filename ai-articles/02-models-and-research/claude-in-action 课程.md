## 0x00 发现宝藏课程

作为长期使用 Claude 的开发者，我在日常编码中已经离不开这个强大的 AI 助手。无论是 Web 端对话，还是 CLI 工具，甚至是各种 IDE 插件的集成，Claude 都能提供相当出色的代码建议和问题解决思路。

但用久了总会遇到一些瓶颈——如何更系统化地使用？有没有官方推荐的“最佳实践”？在一些复杂任务中，怎样让 Claude 真正理解我的完整开发上下文？

于是我开始寻找官方的“标准操作手册”。翻遍了各种技术论坛、博客文章后，**一个偶然的机会，我发现了 Anthropic 官方的学习平台 —— Anthropic Academy**。

看到这个课程后，我感觉就像打游戏找到了攻略。。。。。。

## 0x02 不止是代码助手那么简单

Anthropic Academy 提供的课程覆盖面相当广，从基础入门到专业开发应有尽有：

- **人工智能素养系列**：包括“框架与基础”、“面向教育人群”、“面向学生”三个维度，适合不同背景的学习者
- **Claude API 开发**：专门针对开发者，教你如何构建基于 Claude 的应用程序
- **Claude Code 实操课程**：这就是我们今天重点要讲的
- **模型上下文简介**：理解 Claude 如何处理和利用上下文信息

对我这样的开发者来说，最实用的无疑是 **《Claude Code in Action》** 和 **《Claude 101》** 这两门课。前者专注于命令行工具的高效使用，后者则提供了 Claude 能力的全景概览。

## 0x03 Claude Code in Action：亲身体验

我花了一些时间完成了《Claude Code in Action》的全部内容，并且通过了最终测试（其实就是几道选择题，检验核心概念的掌握程度），成功拿到了结业证书。

<img src="https://cdn.jsdelivr.net/gh/doggaifan/picbed/image-20260204145920213.png" alt="image-20260204145920213" style="zoom:33%;" />

**课程亮点**

1. **结构清晰，层层递进**
   课程从最基本的命令行安装配置开始，逐步深入到高级功能。15 节课程，约 1 小时的总时长，安排得非常紧凑，没有任何冗余内容。
2. **实战导向，即学即用**
   每讲完一个概念，立即配以实际的操作演示。比如讲到“文件操作”时，会现场演示如何让 Claude Code 批量修改多个文件中的特定代码模式。
3. **覆盖高级场景**
   不只是基础操作，课程还包含了 MCP 服务器集成、GitHub 自动化、钩子（hooks）编写等高级主题。这些正是我在日常开发中遇到瓶颈的地方。

**核心收获**

- **上下文管理技巧**：学会了如何使用 `/init` 命令和 `Claude.md` 文件，让 Claude Code 更好地理解我正在处理的项目
- **工作流优化**：掌握了多种热键和命令，可以更流畅地控制与 Claude Code 的交互过程
- **自动化能力**：学会了创建自定义命令，将重复的开发任务自动化

>真正有没有收获暂时保留，反正英语的提升是挺大的，建议大家全程英文观看视频和阅读，手动狗头。

## 0x05 为什么官方的课程如此重要？

在亲自体验了这些课程后，我深刻体会到官方系统学习的价值：

1. **避免“野生”使用习惯**：很多开发者（包括之前的我）都是通过试错来使用 AI 工具，可能会形成一些低效甚至错误的使用习惯。
2. **掌握最新功能**：Claude 在快速迭代，官方课程能确保你学到的是最新的最佳实践。
3. **理解设计哲学**：知道某个功能“怎么用”很重要，但理解“为什么这样设计”更重要，这能帮助你创造性地应用工具。
4. **获得完整视角**：民间教程往往只关注某个具体问题，而官方课程提供了从基础到高级的完整学习路径。

## 0x06 给开发者的学习建议

如果你也打算系统学习 Claude 的高效使用方法，我建议的路径是：

1. **从《Claude 101》开始**：建立对 Claude 能力的整体认知
2. **重点学习《Claude Code in Action》**：这是提升日常开发效率最直接的工具
3. **根据需求选择专业课程**：如果是产品开发，学 Build with Claude API；如果是团队管理，学 Claude for Work。
4. **实践，实践，再实践**：每学一个技巧，立即在自己的项目中尝试应用

Claude 和其他 AI 工具正在快速改变我们的开发方式。与其被动适应，不如主动掌握。这些官方课程就像游戏中的“技能书”，能让你在 AI 时代保持竞争优势。

**从“会用”到“精通”**，可能就差这一个学习平台的距离。
