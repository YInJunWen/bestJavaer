速度太快了！今天 Hermes 宣布可以接入微信了。



Hermes Agent 今天下午刚刚单独面向中国区用户发布了这条消息，而且声明完全是用中文发的。说实话，这种行为方式真的挺圈粉的——相比之下，Hermes 作者对中国区用户的重视程度，比 OpenClaw 的 Peter 那边要好太多了。



技术实现上，Hermes 使用的是腾讯的 **iLink Bot API** 来连接个人微信账号。消息通过长轮询机制传递，因此无需公共端点或 Webhook。微信扫码即可连接，私聊群聊都支持，图片、视频、文件、语音消息全覆盖，长轮询直连，不需要公网 IP。



安装链接在这里：https://hermes-agent.nousresearch.com/docs/user-guide/messaging/weixin



如果你用过腾讯的 QClaw，会发现接入方式是一样的，主要分下面这几个步骤：



1. 通过 iLink Bot API 请求二维码
2. 在终端上显示二维码（或提供网址）
3. 使用微信手机应用扫描二维码
4. 在手机上确认登录
5. 帐户凭据自动保存到 `~/.hermes/weixin/accounts/`



不得不说的是，Hermes 作者在这方面确实很与时俱进——而且这个接入方式还不是他自己想出来的，而是采纳了 contributor 的建议。换句话说，这个项目没有去碰微信的底层协议，是相对安全的做法。



不过这里还是得提醒一下读者朋友们：这毕竟不是微信官方发布的声明，因此我认为依然存在一定风险。就连 Nous Research 自己也发了声明：



“本适配器使用的是腾讯官方的 iLink Bot API，而非逆向工程协议或非官方客户端。不过，如果您担心账号被封禁，绝对值得保持谨慎，并使用小号进行测试！”



如果你想尝鲜试一下，建议先用小号试试水。
